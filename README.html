Yayınlama (GitHub Pages)

  1. Github’da yeni bir repo açın.
  2. Bu dosyaları yükleyin.
  3. Settings → Pages: Deploy from a branch → main / root.
  4. Adrese gidin: https://kullanici.github.io/lokanta-menu/
